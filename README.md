# ДнД: Нож Дануолла

В этом репозитории находится сайт с правилами для моей кампании ДнД.

## Запуск у себя

### Установка зависимостей

```bash
npm install
```

### Разработка

```bash
npm run dev
```

Потом открыть [http://localhost:3000/dnd](http://localhost:3000)

### Размещение
