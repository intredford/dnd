# DnD: Нож Дануолла

Сайт с правилами для моей кампании (реализация отложена на неопределённый срок `:(`).

## Разработка

Установите зависимости:

```bash
npm install
```

Запустите сервер:

```bash
npm run dev
```
## Деплой
При деплое на Github Pages создайте файл .nojekyll в корне проекта.
